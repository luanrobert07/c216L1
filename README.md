# CRUD de Alunos em Python

## 📚 Descrição

Este projeto consiste em um programa desenvolvido em **Python** que executa operações básicas de **CRUD (Create, Read, Update, Delete)** para gerenciamento de alunos de uma faculdade.

O sistema funciona diretamente no **terminal**, permitindo cadastrar, visualizar, atualizar e remover alunos.

Cada aluno possui as seguintes informações:

* Nome
* Email
* Curso (GES, GEC, GET, GEP, etc.)
* Matrícula (gerada automaticamente)

A matrícula é gerada automaticamente como um **número sequencial** pelo sistema.

---

## ⚙️ Funcionalidades

O sistema possui as seguintes operações:

1. **Cadastrar aluno**
2. **Listar alunos cadastrados**
3. **Buscar aluno por matrícula**
4. **Atualizar dados do aluno**
5. **Remover aluno**
6. **Sair do sistema**
